# Next.js Web App

A full-stack web application built with **Next.js** and **MongoDB**. This project utilizes the modern Next.js App Router and includes dynamic routing, API endpoints, and UI styling.

## 🚀 Features
* **Framework:** Next.js powered by React and TypeScript.
* **Database:** MongoDB integration for backend data storage.
* **Styling:** Tailwind CSS configured for responsive design.
* **Routing:** App Router implementation featuring dynamic routes, custom error pages (`error.tsx`, `not-found.tsx`), and loading states.
* **Architecture:** Organized structure with dedicated directories for API routes, modular components, and database models.

## 🛠️ Tech Stack
* Next.js
* TypeScript
* Tailwind CSS
* MongoDB
